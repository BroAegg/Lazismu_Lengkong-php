# Folder Assets/Images

Letakkan file gambar berikut di folder ini:

## Required Images

### Logos
- `logo-lazismu.png` - Logo Lazismu Lengkong (transparent background)
- `logo-muhammadiyah.png` - Logo Muhammadiyah
- `bsi-logo.png` - Logo Bank Syariah Indonesia

### Social Media
- `favicon.png` - Favicon untuk browser tab (32x32 atau 64x64 px)
- `og-image.jpg` - Gambar untuk share di social media (1200x630 px)

### Hero Section
- `hero-image.jpg` - Gambar utama di hero section (opsional, ada placeholder)

### Gallery (Opsional)
- Foto kegiatan penyaluran
- Foto Panti Taman Harapan
- Foto Sekolah Muhammadiyah
- Foto testimoni donatur

## Recommended Format
- PNG untuk logo (dengan transparency)
- JPG untuk foto (compressed)
- WebP untuk performa optimal (opsional)

## Recommended Size
- Logo: max 500KB
- Hero: max 500KB
- Gallery: max 300KB each
